---

## Installation

```bash
git clone https://github.com/lw-108/LW-s-Music-Corner.git
```

1. Open the project in Android Studio  
2. Sync Gradle  
3. Run on an Android device (Android 7.0 or higher)  

---

## Permissions

- READ_EXTERNAL_STORAGE  
- READ_MEDIA_AUDIO (for Android 13+)  

---

## Compatibility

- Android 7.0 (API 24) and above  
- Supports multiple screen sizes
